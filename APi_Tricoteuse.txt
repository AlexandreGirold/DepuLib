Liste des acteurs parlementaires​

Retourne la liste des acteurs parlementaires (députés et sénateurs) avec leurs informations de base.
Path Parameters

    format
    Type: string enum

    Format de la réponse
    values
        csv
        rss
        json

Query Parameters

    page
    Type: number
    min:  
    1

    Numéro de la page à récupérer
    perPage
    Type: number
    min:  
    1

    Nombre d'éléments par page
    select
    Type: array string[] enum

    Champs à inclure dans la réponse, séparés par des virgules
    values
        uid
        senatMatricule
        datePremierAjout
        dateMaj
        dataset
    distinct
    Type: array string[] enum

    Champs pour lesquels les résultats doivent être distincts, séparés par des virgules
    values
        uid
        senatMatricule
        datePremierAjout
        dateMaj
        dataset
    highlightedFields
    Type: array string[]

    Champs à mettre en évidence dans les résultats de recherche, séparés par des virgules. Utilisé avec le paramètre search
    unselect
    Type: array string[] enum

    Champs à exclure de la réponse, séparés par des virgules
    values
        uid
        senatMatricule
        datePremierAjout
        dateMaj
        dataset
    searchLanguage
    Type: string enum

    Langue pour la recherche plein texte
    values
        french
        simple
    searchTable
    Type: string

    Table personnalisée pour la recherche
    prefixSearch
    Type: boolean

    Activer la recherche par préfixe
    search
    Type: string

    Terme de recherche plein texte
    sort
    Type: string

    Critère(s) de tri au format champ.ordre, séparés par des virgules. Champs triables : nom, prenom, dateNais, dateDeces, nombreAmendements, nombreAmendementsAdoptes, nombreQuestions, nombreQuestionsRepondues, nombreMandats, nombreDocumentPublies
    include
    Type: array string[]

    Relations à inclure dans la réponse, séparées par des virgules
    groupBy
    Type: array string[] enum

    Grouper les résultats par un ou plusieurs champs (séparés par des virgules). Champs disponibles : chambre, actif, groupeParlementaireUid, circonscriptionUid
    values
        chambre
        actif
        groupeParlementaireUid
        circonscriptionUid
    uid
    Type: array string[]

    Filtre sur les identifiants uniques (séparés par des virgules)
    chambre
    Type: string enum

    Filtre sur la chambre (AN = Assemblée Nationale, SN = Sénat, CG = Congrès)
    values
        AN
        SN
        CG
        GVT
        AUTRE
    legislature
    Type: number

    Filtre sur la législature
    actif
    Type: boolean

    Filtrer par statut actif/inactif
    slug
    Type: string

    Filtrer les acteurs parlementaires par slug
    groupeParlementaireUid
    Type: string

    Filtrer les acteurs parlementaires par UID de groupe parlementaire
    mandatPrincipalUid
    Type: string

    Filtrer les acteurs parlementaires par UID de mandat principal
    circonscriptionUid
    Type: string

    Filtrer les acteurs parlementaires par UID de circonscription
    nombreAmendements.gte
    Type: number

    Filtrer les acteurs parlementaires ayant au moins ce nombre d'amendements
    nombreAmendements.lte
    Type: number

    Filtrer les acteurs parlementaires ayant au plus ce nombre d'amendements
    nombreAmendementsAdoptes.gte
    Type: number

    Filtrer les acteurs parlementaires ayant au moins ce nombre d'amendements adoptés
    nombreAmendementsAdoptes.lte
    Type: number

    Filtrer les acteurs parlementaires ayant au plus ce nombre d'amendements adoptés
    nombreQuestions.gte
    Type: number

    Filtrer les acteurs parlementaires ayant posé au moins ce nombre de questions
    nombreQuestions.lte
    Type: number

    Filtrer les acteurs parlementaires ayant posé au plus ce nombre de questions
    nombreQuestionsRepondues.gte
    Type: number

    Filtrer les acteurs parlementaires ayant eu au moins ce nombre de questions répondues
    nombreQuestionsRepondues.lte
    Type: number

    Filtrer les acteurs parlementaires ayant eu au plus ce nombre de questions répondues
    nombreMandats.gte
    Type: number

    Filtrer les acteurs parlementaires ayant au moins ce nombre de mandats
    nombreMandats.lte
    Type: number

    Filtrer les acteurs parlementaires ayant au plus ce nombre de mandats
    nombreDocumentsPublies.gte
    Type: number

    Filtrer les acteurs parlementaires ayant publié au moins ce nombre de documents
    nombreDocumentsPublies.lte
    Type: number

    Filtrer les acteurs parlementaires ayant publié au plus ce nombre de documents
